#### 👉 **仓库内容**

***VisitorMS - 访客管理控制系统***

> - 基于树莓派Linux系统开发的访客管理控制应用系统。
> - 系统既有多种用户身份验证功能，又针对不同身份的用户提供不同的应用功能。
> - 涉及多线程、多并发、socket编程、数据加密等功能，设备控制部分基于简单工厂模式开发；
> - 运用ncurses界面库、libcurl网络库、sqlite3数据库、wiringPi驱动库；
> - 项目还包含MJPG-Streamer开源视频方案、安卓APP开发。
